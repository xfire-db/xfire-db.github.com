### SINCLUDE
	SINCLUDE <key> <set-key1> <set-key2> ...
Check whether or not a given list of keys is included in the
set.

### SADD
	SADD <key> <key1> <key2> ...
Add a list of keys to a set.

### SDEL
	SDEL <key> <key1> <key2> ...
Delete a list of keys from a set.

### SCLEAR
	SCLEAR <key>
Clear all keys from set.

### DELETE <key>
	DELETE <key>
Delete the given key.
